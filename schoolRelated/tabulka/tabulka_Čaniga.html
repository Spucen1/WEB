Koncept Python JavaScript
PREMENNÉ Deklarácia:
meno = "Ján"
vek = 25
cena = 19.99
Typy: Dynamické typovanie, automatická detekcia typu 		Deklarácia:
let meno = "Ján";

Typy: Dynamické typovanie, var/let/const
CYKLY For cyklus:
for i in range(5):
    print(i)

zoznam = [1, 2, 3]
for item in zoznam:
    print(item)
While cyklus:
i = 0
while i < 5:
    print(i)
    i += 1
 For cyklus:
for (let i = 0; i < 5; i++) {
    console.log(i);
}

let zoznam = [1, 2, 3];
for (let item of zoznam) {
    console.log(item);
}
While cyklus:
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
FUNKCIE Definícia:
def pozdrav(meno):
    return f"Ahoj {meno}!"

sprava = pozdrav("Ján")
print(sprava)
Vlastnosti: Kľúčové slovo def, odsadenie definuje blok 		Definícia:
function pozdrav(meno) {
    return `Ahoj ${meno}!`;
}

const pozdrav2 = (meno) => {
    return `Ahoj ${meno}!`;
};
Vlastnosti: function alebo arrow funkcie =>
